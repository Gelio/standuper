# Standuper

[See it live](https://gelio.github.io/standuper/)

Standuper is a web application to facilitate daily stand-up meetings for teams.

- Timer (customizable duration)
- A list of team members
- A button to shuffle the speaking order
- Data is stored in `localStorage` so you don't have to re-enter it every time

## Development

Run `bun run dev` to start the local dev server.
